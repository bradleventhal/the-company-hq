## DM Campaign Strategy

### Timing
- **8-10am PT:** West coast US devs
- **9-11am ET:** East coast US devs
- **2-4pm CET:** European devs
- **Evening:** Follow up with anyone who engaged

### Approach
1. **Personalize opening** - Reference their work
2. **Quick pitch** - One sentence what it is
3. **Demo link** - Always include
4. **Light ask** - "Curious what you think" not "please share"

### Success Metrics
- **10%+ reply rate** = Good (2+ replies from 20 DMs)
- **20%+ reply rate** = Great (4+ replies)
- **Any reshare** = Viral potential

### Follow-up Rules
- If they reply: Thank them, answer questions
- If they try it: Ask for screenshot
- If they share: Retweet with gratitude
- If no reply in 48hr: Move on (don't double DM)

### If they ask "Why RPG?"
```

RPG elements (XP, leveling, quests) make progress visible and rewarding. 
Turns monitoring into something you actually want to do.

Plus pixel art aesthetic = instant nostalgia factor.
```

```
Next.js + TypeScript for the app
```
