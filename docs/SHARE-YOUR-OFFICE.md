## Recording Tools

### Mac (Built-in)

**Screenshot:**
- `Cmd + Shift + 4` → Select area
- `Cmd + Shift + 4 + Space` → Capture window

**Screen Recording:**
- `Cmd + Shift + 5` → Record selected area
- Quick 10-15 second clips

### Windows

**Screenshot:**
- `Win + Shift + S` → Snip tool

**Screen Recording:**
- `Win + G` → Xbox Game Bar → Record

### Linux

**Screenshot:**
- `gnome-screenshot` or `scrot`

**Screen Recording:**
- `SimpleScreenRecorder` or `Kazam`

### Cross-Platform Tools

**Kap** (Mac) - https://getkap.co
- Easy GIF creation
- Built-in editing
- Export straight to Twitter size

**OBS Studio** (All) - https://obsproject.com
- Full control
- High quality
- More complex setup

## Pro Sharing Tips

### 1. Show Activity, Not Static

Instead of just the dashboard, show:
- Agents switching status (working → idle)
- Quests being approved
- Accomplishments appearing
- Chat messages flowing

**Motion = engagement**

### 2. Tell a Story

Instead of: "Here's OpenClawfice"
Try: "I was spending 2 hours/day managing agents. Now it's 5 minutes."

**Problem → Solution = relatability**

### 3. Include a Call-to-Action

Always end with:
- "Try the demo: [link]"
- "What do you think?"
- "How do you manage your agents?"

**Engagement = virality**

### 4. Respond to Comments

When people reply:
- Answer questions
- Share more details
- Link to docs

**Engagement = algorithm boost**

---

## What NOT to Share

❌ **Sensitive agent conversations** (private data, API keys, credentials)
❌ **Production issues** (outages, critical bugs - save for after fixing)
❌ **Negative posts** ("OpenClawfice sucks because..." - open a GitHub issue instead)
❌ **Spam** (same post to 10 subreddits in 5 minutes - pace it out)
